> [!IMPORTANT]
> OP Auto Clicker is designed for **Windows**.  
> If you are on **macOS** or **Linux**, this specific build might not run natively. Check out alternatives like **[AutoClicker-Desktop](https://github.com/autoclicker/autoclicker)** for cross-platform needs.

## Why use OP Auto Clicker?

*   **Ultra Fast:** Faster than your jitter-clicking cousin.
*   **Customizable:** Set your own intervals, from hours to milliseconds.
*   **Simple UI:** No rocket science here, just open and click.
*   **Open Source:** No hidden miners, just pure clicking power.

## Installation

1. Go to the [Releases](https://github.com/jiaoyanming0-bot/OPAutoClicker/releases/latest) page.
2. Download the `.exe` file.
3. Run it (no installation required, it's portable!).
4. Set your hotkeys and let the clicking begin.

## Frequently Asked Questions (FAQ)

<details>
  <summary><strong>Will I get banned in games?</strong></summary>
  <br>

  Most games (especially on Roblox) don't ban for simple auto-clickers. However, some competitive games with strict anti-cheats might detect "inhumanly" consistent click patterns. Use a bit of "Randomize" delay if the game feels suspicious!
</details>

<br>

<details>
  <summary><strong>Is it a virus?</strong></summary>
  <br>

  Windows Defender loves to flag small, unsigned tools like this as "Trojan" or "Malware". It's a false positive because the app "takes control" of your mouse.  
  Since this is **open-source**, you can literally read the code and build it yourself if you don't trust the .exe.
  
  Check the code here:
  https://github.com/jiaoyanming0-bot/OPAutoClicker
</details>
